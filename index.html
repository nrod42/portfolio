TV/Movie Tracker

The TV Show/Movie Tracker is a React-based web application developed using React/JS, Bootstrap/CSS, and HTML. It leverages the TMDB API to fetch and display TV shows and movies, providing users with an immersive browsing experience. The project showcases strong technical skills in front-end development, including interactive media cards, personalized watch lists, and real-time search functionality.

On the backend, the application incorporates Node.js, Express, and MongoDB. User watch lists are securely stored in MongoDB, allowing users to save and organize their favorite shows and movies. The implementation includes user registration and login functionality, utilizing bcrypt for password hashing and JSON Web Tokens for secure authentication and authorization. This ensures privacy and data protection for user accounts.

Blog Site

The Blog Site project is a dynamic and interactive full-stack blogging platform that offers seamless content management and engagement. Built with React on the frontend, the application provides a responsive and intuitive user interface for effortless blog creation and management. Users can easily create, read, update, and delete blog posts using rich text editing capabilities and image uploading. Additionally, the site provides personalized profiles for users to conveniently manage their information and access their posts.

On the backend, the site leverages MongoDB for data storage and integrates Express and Node.js to handle server-side functionalities. MongoDB's robust text indexing feature is utilized to provide efficient content discovery through the platform's search functionality. User registration and authentication are implemented using bcrypt for password hashing and JSON Web Tokens for secure user authorization, ensuring privacy, data protection, and secure access to the platform.

Shopping Site

The Shopping Site project is a responsive product showcase developed with React and Bootstrap. Users can browse and filter products based on categories, view detailed information on individual product pages, and add items to their cart. The dynamic off-canvas cart provides real-time quantity updates, ensuring a seamless and convenient shopping experience. Although the site does not support actual purchases, it provides a streamlined purchasing experience by allowing users to review their cart at any point.
