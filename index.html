The Blog Site project is a dynamic and interactive full-stack blogging platform that offers seamless content management and engagement. Built with React on the frontend, the application provides a responsive and intuitive user interface for effortless blog creation and management. Users can easily create, read, update, and delete blog posts using rich text editing capabilities and image uploading. Shopping Site

The Shopping Site project is a responsive product showcase developed with React and Bootstrap. Users can browse and filter products based on categories, view detailed information on individual product pages, and add items to their cart. The dynamic off-canvas cart provides real-time quantity updates, ensuring a seamless and convenient shopping experience. Although the site does not support actual purchases, it provides a streamlined purchasing experience by allowing users to review their cart at any point.

Resume Builder

The application features a responsive navigation system, interactive forms, and a card-based layout for an intuitive user experience. The integration of PDF download functionality enables easy generation and sharing of professional resumes.
